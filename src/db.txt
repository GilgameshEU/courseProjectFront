1-Таблица "users" - содержит информацию о зарегистрированных пользователях
id (int, primary key)
username (varchar)
password (varchar)
is_admin (boolean)
is_blocked (boolean)
created_at (timestamp)
2-Таблица "collections" - содержит информацию о коллекциях
id (int, primary key)
user_id (int, foreign key)
name (varchar)
description (text)
theme (varchar)
image_url (varchar)
created_at (timestamp)
updated_at (timestamp)
3-Таблица "items" - содержит информацию об айтемах
id (int, primary key)
collection_id (int, foreign key)
name (varchar)
description (text)
tags (varchar)
field_1 (varchar)
field_2 (varchar)
field_3 (varchar)
field_4 (text)
field_5 (text)
field_6 (text)
field_7 (date)
field_8 (date)
field_9 (date)
field_10 (boolean)
field_11 (boolean)
field_12 (boolean)
created_at (timestamp)
updated_at (timestamp)
4-Таблица "likes" - содержит информацию о лайках пользователей на айтемы
id (int, primary key)
user_id (int, foreign key)
item_id (int, foreign key)
created_at (timestamp)
5-Таблица "comments" - содержит информацию о комментариях к айтемам
id (int, primary key)
user_id (int, foreign key)
item_id (int, foreign key)
comment_text (text)
created_at (timestamp)
